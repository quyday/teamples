CSS – Biến website thành một tác phẩm đẹp mắt

Tìm hiểu về CSS - ngôn ngữ tạo kiểu cho trang web, giúp website trở nên đẹp mắt và chuyên nghiệp.

CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ tạo kiểu được sử dụng để định dạng và trình bày các trang web. Nếu HTML là bộ xương của trang web, thì CSS chính là lớp da, quần áo và trang điểm giúp trang web trở nên đẹp mắt và chuyên nghiệp.

Tại sao CSS lại quan trọng?

CSS là thành phần không thể thiếu trong bộ ba công nghệ web (HTML, CSS, JavaScript). Nó giúp bạn:

  • Kiểm soát bố cục và thiết kế của trang web
  • Tạo giao diện người dùng hấp dẫn và dễ sử dụng
  • Đảm bảo trang web hiển thị nhất quán trên các thiết bị khác nhau
  • Tách biệt nội dung (HTML) và cách trình bày (CSS), giúp code dễ bảo trì hơn

Cách sử dụng CSS

1. Cú pháp CSS cơ bản

selector {
    thuộc_tính: giá_trị;
    thuộc_tính_khác: giá_trị_khác;
}

Ví dụ:

h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

p {
    color: #333;
    font-size: 16px;
    line-height: 1.5;
}

2. Ba cách để thêm CSS vào trang web

a) Inline CSS
<p style="color: red; font-size: 18px;">Đây là văn bản màu đỏ.</p>
b) Internal CSS
<head>
    <style>
        p {
            color: red;
            font-size: 18px;
        }
    </style>
</head>
c) External CSS (Cách tốt nhất)
/* File style.css */
p {
    color: red;
    font-size: 18px;
}

/* Trong file HTML */
<head>
    <link rel="stylesheet" href="style.css">
</head>

3. Các loại selector trong CSS

a) Element Selector
p {
    color: red;
}
b) Class Selector
.highlight {
    background-color: yellow;
}

<p class="highlight">Đoạn văn được highlight</p>
c) ID Selector
#header {
    background-color: black;
    color: white;
}

<div id="header">Đây là header</div>

Các thuộc tính CSS phổ biến

1. Thuộc tính văn bản

  • color: Màu chữ
  • font-family: Kiểu font
  • font-size: Kích thước font
  • font-weight: Độ đậm của font
  • text-align: Căn chỉnh văn bản
  • line-height: Chiều cao dòng

2. Thuộc tính box model

  • margin: Khoảng cách bên ngoài phần tử
  • padding: Khoảng cách bên trong phần tử
  • border: Viền của phần tử
  • width: Chiều rộng
  • height: Chiều cao

3. Thuộc tính background

  • background-color: Màu nền
  • background-image: Hình nền
  • background-repeat: Lặp lại hình nền
  • background-position: Vị trí hình nền

CSS nâng cao

1. Responsive Web Design

Responsive Web Design giúp trang web hiển thị tốt trên mọi thiết bị (desktop, tablet, mobile). Media queries là công cụ chính để thực hiện điều này:

/* Áp dụng cho màn hình có chiều rộng tối đa 768px */
@media (max-width: 768px) {
    .container {
        width: 100%;
    }
    
    .menu {
        display: none;
    }
}

2. Flexbox và Grid

Flexbox và Grid là hai công cụ mạnh mẽ để tạo bố cục linh hoạt và phức tạp:

/* Flexbox */
.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

/* Grid */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 20px;
}

Bắt đầu học CSS như thế nào?

Để bắt đầu học CSS, bạn nên:

  1. Nắm vững HTML trước (vì CSS hoạt động dựa trên HTML)
  2. Học cú pháp cơ bản của CSS
  3. Thực hành với các ví dụ đơn giản
  4. Tìm hiểu về box model và cách CSS xử lý bố cục
  5. Học responsive design để trang web hiển thị tốt trên mọi thiết bị
  6. Khám phá các framework CSS như Bootstrap, Tailwind CSS

Kết luận

CSS là công cụ không thể thiếu để tạo ra các trang web đẹp mắt và chuyên nghiệp. Nó giúp bạn kiểm soát hoàn toàn cách trình bày nội dung HTML. Với HTML làm nền tảng và CSS làm lớp trang trí, bạn đã có hai phần quan trọng trong bộ ba công nghệ web cơ bản. Hãy tiếp tục hành trình học web của mình với JavaScript để tạo ra các trang web động và tương tác!
